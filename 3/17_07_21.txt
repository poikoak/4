1. В задаче с текстовым редактором добавить возможность использовать MDI-окна вместе с вкладками. В настройках
предоставить пользователю выбрать способ, который он хочет. Добавить контекстное меню, там, где это удобно.

2. При нажатии мышью в пустом месте окна добавляется новый шарик, который двигается по таймеру и отскакивает
от стен. При нажатии мышью на шарик, он исчезает. Шарики не вылетают за границы окна.
Задание на +1 к экзамену: шарики упруго отскакивают при столкновении друг с другом.

3. Разработать программу-телефонный справочник, которая имеет следующие функции:
- добавление контакта в отдельном окне
- просмотр контакта в отдельном окне
- редактирование контакта в отдельном окне
- удаление контакта
- при добавлении и редактировании контакта пользователь может добавить фото
- поля контакта: ФИО, телефон, адрес, описание, фото, категория контакта
- при выходе из программы все контакты сохраняются, при входе они загружаются
- +1 за шифрование контактов и ввод пароля

4. Один из проектов разработать с использованием Git (прислать ссылку на репозиторий)
